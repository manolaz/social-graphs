# social-graphs
Exploid all pairs connected COMMUNITIES  (CLIQUE) in a undirected (Biconnected) graph.

## Install Libraries

```bash
pip3 install py3plex
```

## Run EXAMPLE
```bash
python3 example/detector.py
```
